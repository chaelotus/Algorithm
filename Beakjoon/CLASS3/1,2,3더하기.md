## 아쉬운점

문제를 보고도 도저히 어떤 식으로 문제를 풀어야하는지 감이 오지 않았다.  
검색해보니 해당 문제를 점화식을 구하여 dp를 이용하여 푸는 것을 알 수 있었다.  
문제 풀이는 아래 링크에서 참고하였다.

- https://ymg5218.tistory.com/144

정리해보면 n=1일때 1개, n=2일때 2개, n=3일때 4개가 나오는데  
n=4일때 1이 제일 마지막으로 오는 수, 2가 제일 마지막으로 우는 수, 3이 제일 마지막으로 오는 수 이렇게 나누어보면 규칙이 보인다.  
1이 제일 마지막으로 오는 수를 보면 n=3일때의 모든 방법에서 뒤에 1을 더한 꼴과 같다.  
2가 제일 마지막으로 오는 수를 보면 n=2일때 모든 방법에서 뒤에 2를 더한 꼴과 같고  
1이 제일 마지막으로 오는 수를 보면 n=1일때 모든 방법에서 뒤에 3을 더한 것과 같다.  
따라서 우리는 dp[n](n>=4)가 1,2,3의 합으로 n을 나타내는 경우의 수라면,  
dp[n] = dp[n-1]+dp[n-2]+dp[n-3] 으로 나타낼 수 있다.

## 다른 사람이 푼 코드

```py
T = int(input())

def dpOp(n):
    if n ==1:
        return 1
    elif n==2:
        return 2
    elif n==3:
        return 4
    else:
        return dpOp(n-1)+dpOp(n-2)+dpOp(n-3)

for _ in range(T):
    n = int(input())
    print(dpOp(n))
```
