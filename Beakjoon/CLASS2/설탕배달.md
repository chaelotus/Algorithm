## 아쉬운 점

문제의 규칙을 코드로 작성하지 못하였다.  
5로 나누는 것 까지는 생각을 하였지만 5와 3을 조합해서 나눠 담는 방법을 어떻게 처리해야할지 감을 잡지 못하였다.  
아래 코드처럼 5로 나누어 떠어지지 않으면 3을 빼고 봉지를 하나 더하고 이런 방법을 생각하지 못하였다. 다른 사람의 풀이를 보니 새삼 너무 쉬워보인다.

## 다른 사람이 푼 코드

```py
N = int(input())
count = 0

while N>=0:
    if N % 5 == 0:
        count+=N//5
        print(count)
        break

    N-=3
    count+=1
else:
    print(-1)
```
