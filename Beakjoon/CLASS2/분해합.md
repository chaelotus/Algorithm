### 아쉬운 점

문제를 잘 이해하지 못하였고 깊이 생각하지 못한것 같다.
다른 사람의 코드를 보니 조금만 더 고민해보면 되는 문제였던 것 같다.

## 다른 사람 코드

```py
n = int(input())

for i in range(1, n+1):
    num = sum(map(int, str(i)))
    #분해합 = 생성자 + 생성자 각 자릿수 합
    num_sum = i + num
    if num_sum == n:
        print(i)
        break
    if i== n:
        print(0)
```
