## 문제 접근 방법
n의 최대값이 10! 값이므로 for문으로 1 부터 10까지 반복하면서 1만큼 커진 값을 계속 곱하면서 어느순간 n의 값보다 클 때 반복문을 빠져나와 i에서 1을 빼준 값을 반환하였다.

</br>

### 알게된 점

</br>

## 코드
```
function solution(n) {
  let answer = 1;
  let i;
  for(i=1; i<=10; i++){
      answer*=i;
      if(answer>n)
        break;
  }
  return --i;
}
```
