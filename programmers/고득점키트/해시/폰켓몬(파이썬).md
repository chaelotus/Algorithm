## 1. 문제 요약 (내 언어로)

- 핵심 요구: 종류가 다른 포켓몬을 N/2개 를 가져가되, 최대한 많은 종류로 가져가기.
- 제한 조건: nums 배열은 무조건 짝수로 줌.

## 2. 접근 방법 (사고 과정)

- 처음 떠올린 방법: 처음에는 n/2개를 고르는 모든 경우의 수를 다 확인해야하나 했는데 해시를 사용하자라는 생각을 한 뒤, nums를 객체화로 만들어 해당 길이와 len(nums)/2 를 비교하여 답을 도출.
- 왜 이 방법을 선택했는지: 해시를 사용하기 위해
- 시간복잡도: O(n)

## 4. 코드 (최종)

```py
def solution(nums):

    obj = {}
    n_2 = int(len(nums) / 2)

    for i in nums:
        if i in obj.keys():
            obj[i]+=1
        else:
            obj[i] = 1

    if len(obj)>= n_2:
        return n_2

    return len(obj)
```

다른 사람 풀이를 보니 내가 작성한 코드보다 훨씬 더 간단하게 작성할 수 있었다.  
먼저 set 자료형으로 for문을 대체할 수 있다.  
set은 수학에서 이야기하는 집합과 비슷하다.  
순서가 없고, 집합안에서는 unique한 값을 가진다. 중괄호를 사용하는 것은 dictionary와 비슷하지만 key가 없다.

```py
s = set([1,3,5,7])

>>> {1,3,5,7}
```

그리고 이후 if문을 통해 답을 도출할때 min을 사용하여 더 간단히 작성할 수 있다. 이러한 내용을 토대로 다시 코드를 작성하면 아래와 같다.

```py
def solution(nums):
    return min(len(set(nums)),len(nums)//2)
```
