## 📌 문제 핵심 아이디어
- 이 문제는 각 카테고리별로 안 입는 선택지는 있지만, 하지만 최소 하나라도 걸쳐야 한다는 포인트를 인지해야 함.  

## ⚠️ 내가 놓친 부분 / 틀린 이유
- 공식을 찾아내지 못함. 

## 💡 핵심 Python 문법/팁
- math.prod() : 인자로 받은 iterable(리스트, 딕셔너리 값 등)의 모든 요소를 곱함 

```py
import math
result = math.prod([3, 2])
```

### 코드

```python
import math

def solution(clothes):
    answer = 1

    clothes_dict = {}

    for cloth in clothes:
        if cloth[1] in clothes_dict:
            clothes_dict[cloth[1]] +=1
        else:
            clothes_dict[cloth[1]] = 2
    
    answer = math.prod(clothes_dict.values()) -1

    return answer
```

### 아쉬운 부분

1. clothes_dict[cloth[1]] = 2 ⇒ 안입는 선택지까지 생각하여 2로 초기화했는데 가독성이 떨어짐

```py
answer = math.prod(count + 1 for count in clothes_dict.values()) -1
```

2. `Counter`를 활용

```py
from collections import Counter

# clothes에서 카테고리 (clothes[1])만 뽑아서 바로 개수를 세어줌

# 예 : Counter({’header’:2, ‘eyewear’:1})

clothes_counter = Counter(clothes[1] for cloth in clothes)

return math.prod(count+1 for count in clothes_counter.values()) - 1
```

            clothes_dict[cloth[1]]+=1

            clothes_dict[cloth[1]]=1
    
    answer = math.prod([v+1 for v in clothes_dict.values()])-1
