## 4. 코드 (최종)

1. hash 활용

```py
def solution(participant, completion):
    answer = ''

    hashDict = {}
    sumHsh = 0

    for part in participant:
        hashDict[hash(part)] = part
        sumHsh+=hash(part)

    for com in completion:
        sumHsh-= hash(com)

    answer = hashDict[sumHsh]

    return answer
```

`hash()`는 데이터를 빠르게 찾기 위해 고유한 정수값으로 변환하는 함수이다.  
값이 똑같으면 동일한 정수값이 나온다. 그래서 sumHash에 participant의 값을 모두 더하고 completion의 값을 뺴고 남은 값으로 hashDict를 찾으면 풀 수 있다.

2. sort 활용

```py
def solution(participant, completion):
    answer = ''

    participant.sort()
    completion.sort()

    for i in range(len(completion)):
        if participant[i] != completion[i]:
            print(participant[i])
            return participant[i]

    return participant[-1]
```

두개의 배열을 `sort()`하여서 completion의 길이만큼 for문을 돈다.  
이후 같지 않은걸 발견하면 해당 값을 return하고 for문을 돌때까지 찾지못하였으면 제일 마지막 값을 return한다.
